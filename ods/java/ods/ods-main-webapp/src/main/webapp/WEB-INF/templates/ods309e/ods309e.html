主題基本功能說明

(1)下載電子檔

可下載目前瀏覽主題的相關內容之PDF檔。

(2)下載分析檔

可下載主題Tableau資料分析檔案，民眾可於Tableau官方網站下載Tableau Reader，以進一步進行資料分析及操作。 http://www.tableausoftware.com/products/reader

(3)主題訂閱

依主題內容性質，將提供多種不同的訂閱方案，民眾可選擇有興趣之方案加以訂閱，當主題資料達到訂閱方案的條件時，本網站將以電子郵件的方式通知以便民眾追縱最新分析報告。

(4)我要評分

民眾可依個人喜好對於每個主題進行評分，本網站也將參考民眾回饋之評分結果調整提供的主題內容，以更符合民眾對主題分析之需求。

(5)瀏覽主題其他版本

電子發票智慧好生活網提供之主題內容依不同性質將會定期更新，本網站將會保存歷次更新版本，民眾若想查看該主題其他版本時，可於此下拉選單選擇版本。

(6)轉貼分享

民眾可將主題內容轉貼分享至Facebook、Google+、Twitter。

素材及案例功能說明

(1)素材及案例類型

每個主題可能包含一至多個素材及案例，而素材及案例共有圖片、資料集以及PDF三種類型。
  • 圖片：可下載另存成PNG檔
  • 資料集：可下載該資料集之CSV檔
  • PDF：可下載另存成PDF檔

(2)素材及案例資料瀏覽

每個素材及案例都可按下資料瀏覽以便瀏覽便詳細資訊，圖片及PDF類型可看到完整的資訊，資料集則可進一步進行線上之視覺化分析功能。

(3)資料集視覺化分析功能

當素材及案例為資料集時，本網提供兩種資料呈現方式：
  • 表格：以表格的方式呈現資料集內容，資料內容預設以每頁100筆顯示，民眾可自行於如圖片中標示「1」的地方調整每頁顯示之筆數與範圍。民眾亦可如圖片中標示「2」的地方設定資料篩選條件，設定完成後表格僅顯示符合篩選條件之資料。
  • 圖形：支援「視覺化資料圖形瀏覽，民眾可於右方區域自訂「圖形類型」、「X軸」、「Y軸」後，網頁將依目前「表格」頁籤中顯示之資料，繪製對應之統計圖形。
    • 圖形類型：支援「折線與散佈圖」、「折線圖」、「散佈圖」、「橫條圖」、「長條圖」、「圓餅圖」等六種視覺化資料統計圖形
    • X軸：設定統計圖型之X軸欄位，建議選擇質性資料類型欄位，如地區、性別等。
    • Y軸：設定統計圖型之Y軸欄位，建議選擇量化資料類型欄位，如金額、數量等。
常見問題
  • 問：為什麼圖片X軸看起來很擠？讓我看不清楚X軸文字？
  • 答：因為智慧好生活提供的資料集種類繁多，當資料集資料量較多時，就有可能有此狀況。建議您善用「搜尋資料」或「過濾條件」等功能，篩選出有興趣的資料條件，縮小資料筆數，就可以改善這個問題。若「搜尋資料」或「過濾條件」功能無法滿足您的需求，建議您參考「下載分析檔」說明，透過Tableau Reader開啟分析檔，可對主題資料做更進一步的資料操作。
