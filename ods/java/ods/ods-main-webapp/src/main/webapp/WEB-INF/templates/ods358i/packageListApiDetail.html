API2：

1.API1可用來查詢所有主題資訊。

2.API2可加入查詢條件，查詢出相關條件主題，查詢修件說明如下：

{packageName}
主題名稱		 輸入主題名稱關鍵字當查詢條件，如名稱有中文字請先使用URLEncoder.encode(’中文’, ’UTF-8’)進行編碼。
{packageTags}
分類標籤		 輸入分類標籤當查詢條件，輸入格式為{{標籤1,標籤2,...}}，如標籤有中文字請先使用URLEncoder.encode(’中文’, ’UTF-8’)進行編碼。
{resourceFileExts}
檔案格式		 輸入檔案格式當查詢條件，檔案格式有image,pdf,dataset三種，輸入格式為{{檔案格式1,檔案格式2,...}}。

3.回傳結果說明：

status 00 查詢成功／01 查無資料／99 查詢失敗
result (1)id:主題ID
(2)name:主題名稱
(3)description:主題說明
(4)url:主題連結
(5)imageUrl:主題圖示連結
