●特定主題API說明

主題發佈版本列表API：

主題版本之素材列表API：

主題版本之素材下載路徑API：

1.主題發佈版本列表API：列出現行所有主題資訊 ，回傳結果說明如下：

status 00 查詢成功／01 查無資料／99 查詢失敗
result (1)packageId:主題ID
(2)publishedVersions:已發佈的主題版本，包含下列資訊：
　i.ver:主題版本
　ii.name:主題版本名稱
　iii.description:主題版本說明

2.主題版本之素材列表API：列出特定主題下之個別版本之所有素材資訊，回傳結果說明如下：

status 00 查詢成功／01 查無資料／99 查詢失敗
result (1)packageId:主題ID
(2)packageVer:主題版本
(3)resources:主題中包含的素材，包含下列資訊：
　i.id:素材ID
　ii.ver:素材版本
　iii.name:素材版本名稱
　iv.description:素材版本說明
　v.type:素材格式，素材分為png/pdf/csv三種格式
　vi.url:素材下載連結

3.主題版本之素材下載路徑API：列出特定主題下之個別版本之所有素材下載路徑資訊，回傳結果說明如下：

status 00 查詢成功／01 查無資料／99 查詢失敗
result (1)packageId:主題ID
(2)packageVer:主題版本
(3)resourcesDownloadList:素材下載路徑資訊，包含下列資訊：
　i.url:素材下載連結
